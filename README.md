# 💬 Zakayo Chat

**Mazungumzo Ya Kweli · Salama · Ya Haraka**

App ya gumzo (chat) kamili inayofanya kazi kwenye simu na kompyuta bila seva ya nje. Weka kwenye GitHub Pages bure na itumike na watu wote duniani kote!

---

## 🚀 Jinsi ya Kuweka kwenye GitHub (Hatua kwa Hatua)

### Hatua 1 — Fungua GitHub
Nenda: **https://github.com** → Ingia au Jisajili bila malipo.

### Hatua 2 — Tengeneza Repository Mpya
1. Bonyeza kitufe cha **`+`** juu kulia → **"New repository"**
2. Jaza:
   - **Repository name:** `zakayo-chat` (au jina lolote unalotaka)
   - **Description:** `Zakayo Chat – App ya mazungumzo`
   - Chagua **Public** ✅
   - Angalia kisanduku cha **"Add a README file"** ✅
3. Bonyeza **"Create repository"**

### Hatua 3 — Pakia Faili
Kwenye ukurasa wa repository yako:
1. Bonyeza **"Add file"** → **"Upload files"**
2. Buruta (drag & drop) faili hizi 4 ZOTE:
   ```
   ✅ index.html
   ✅ manifest.json
   ✅ sw.js
   ✅ README.md
   ```
3. Scroll chini → andika "Pakia Zakayo Chat v2.1" → Bonyeza **"Commit changes"**

### Hatua 4 — Washa GitHub Pages
1. Bonyeza **"Settings"** (mwisho wa menyu juu)
2. Upande wa kushoto, bonyeza **"Pages"**
3. Chini ya **"Branch"**, chagua:
   - Branch: **`main`** (au `master`)
   - Folder: **`/ (root)`**
4. Bonyeza **"Save"**
5. Subiri dakika 1-2...

### Hatua 5 — Pata Link Yako!
GitHub itakupa link kama hii:
```
https://JINA_LAKO.github.io/zakayo-chat/
```

**🎉 Hiyo ndiyo link ya kutumia na kutuma WhatsApp!**

## 📱 Jinsi ya Kufungua kwenye Simu

### Android:
1. Fungua Chrome → nenda kwenye link yako
2. Bonyeza menyu (⋮) → **"Add to Home screen"**
3. Bonyeza **"Add"** → App itaonekana kwenye skrini ya nyumbani kama app ya kawaida!

### iPhone (iOS):
1. Fungua Safari → nenda kwenye link yako
2. Bonyeza kitufe cha kushiriki (↑) chini ya skrini
3. Scroll chini → **"Add to Home Screen"**
4. Bonyeza **"Add"** → Done! ✅
