💾 Local Storage

📚 Somo 1 — Local Storage ni Nini?

Local Storage ni sehemu ya browser inayohifadhi data — hata ukifunga browser data inabaki!


// Hifadhi data
localStorage.setItem("jina", "Zakayo");

// Soma data
localStorage.getItem("jina");

// Futa data
localStorage.removeItem("jina");

// Futa yote
localStorage.clear();

⚙️ Somo 3 — Hifadhi Mipangilio

Hifadhi mipangilio ya mtumiaji — rangi, lugha, na zaidi!


// Hifadhi object
const mipangilio = {rangi: "nyekundu", lugha: "Kiswahili"};
localStorage.setItem("mipangilio", JSON.stringify(mipangilio));

// Soma object
const data = JSON.parse(localStorage.getItem("mipangilio"));
