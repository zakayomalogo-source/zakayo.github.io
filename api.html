💾 Local Storage

📚 Somo 1 — Local Storage ni Nini?

Local Storage ni sehemu ya browser inayohifadhi data — hata ukifunga browser data inabaki!


// Hifadhi data
localStorage.setItem("jina", "Zakayo");

// Soma data
localStorage.getItem("jina");

// Futa data
localStorage.removeItem("jina");

// Futa yote
localStorage.clear();

🌐 API na Fetch

📚 Somo 1 — API ni Nini?

API ni njia ya tovuti yako kuzungumza na tovuti nyingine — kupata data kama hali ya hewa, habari, picha na zaidi!


// Fetch — kuomba data kutoka API
fetch("https://api.example.com/data")
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.log(error));

Maana yake:


✅ fetch() — omba data

✅ .then() — ukipata jibu fanya hivi

✅ .json() — badilisha jibu kuwa data

✅ .catch() — kama kuna kosa fanya hivi

😄 Somo 2 — Jokes API

Tutapata mzaha kutoka API ya kweli mtandaoni!


fetch("https://official-joke-api.appspot.com/random_joke")
  .then(res => res.json())
  .then(data => {
    console.log(data.setup);
    console.log(data.punchline);
  });

Bonyeza kitufe kupata mzaha kutoka API! 😄

🔢 Somo 3 — Numbers API

Pata ukweli wa kuvutia kuhusu nambari yoyote!


fetch(http://numbersapi.com/${nambari})
  .then(res => res.text())
  .then(data => console.log(data));

🎮 Somo 4 — Pokemon API

Pata taarifa za Pokemon kutoka API — inatumiwa na mamilioni ya watu!


fetch("https://pokeapi.co/api/v2/pokemon/pikachu")
  .then(res => res.json())
  .then(data => {
    console.log(data.name);
    console.log(data.height);
  });

⚡ Somo 5 — Async/Await

Njia ya kisasa zaidi ya kuandika Fetch — rahisi kusoma!


// Njia ya zamani — .then()
fetch(url).then(res => res.json()).then(data => ...);

// Njia ya kisasa — async/await
async function pataDana() {
  const res = await fetch(url);
  const data = await res.json();
  console.log(data);
}

Bonyeza kitufe kuona async/await ikifanya kazi!
